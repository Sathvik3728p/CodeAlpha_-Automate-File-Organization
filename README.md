# 📂 Move `.jpg` Files (Python)

## 📌 Overview
This project is a simple Python automation script that organizes image files. It scans a source folder, finds all `.jpg` files, and moves them into a specified destination folder. A practical way to save time and reduce repetitive manual work.

---

## ⚙️ Features
- Detects `.jpg` files in the source folder  
- Creates the destination folder if it doesn’t exist  
- Moves files safely using Python’s `shutil` module  
- Prints a success message once the operation is complete  

---

## 🛠️ Concepts Used
- **os module** → directory handling  
- **shutil module** → file moving operations  
- **For loop** → iterate through files  
- **If condition** → check file extensions  

---

## 🚀 How to Run
Clone the repository:
```bash
git clone https://github.com/yourusername/move-jpg-files.git
```

Navigate to the project folder:
```bash
cd move-jpg-files
```

Run the script:
```bash
python move_jpg_files.py
```
